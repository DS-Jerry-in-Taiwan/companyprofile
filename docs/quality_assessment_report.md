| 測試文字 | 預期範圍 | 實際結果 | 狀態 |
|---------|----------|----------|------|
| "Hello World 測試文字" | 5-10 tokens | 正確 | ✅ |
| 中英混合文字 | 按比例計算 | 合理 | ✅ |
| 長文字 | 合理估算 | 合理 | ✅ |

**Token 估算公式**:
```
estimated_tokens = chinese_chars / 1.5 + other_chars / 4
```
- 中文：約 1.5 個字 = 1 token
- 英文：約 4 個字元 = 1 token
