# Phase 4 - 驗證清單 (Validation Checklist)

**階段**: Phase 4 - 安全防護與內容審核 (Risk Control)

---

## ✅ 品質檢查項目

### 1. 基礎設施 (INFRA)
- [ ] `bleach` 庫是否已安裝？
- [ ] 敏感詞與競品清單路徑配置正確？

### 2. 安全策略 (ARCH)
- [ ] 是否繪製了風控檢查的序列圖？
- [ ] `RiskStatus` 列舉包含 (PENDING, APPROVED, REJECTED)？

### 3. 風控功能 (CODER)
- [ ] 敏感詞正則表達式掃描器可運作？
- [ ] 競品名稱過濾器可運作 (如: 104 -> ***)？
- [ ] HTML 消毒模組是否成功清洗掉非白名單標籤？

### 4. 交付品質 (ANALYST)
- [ ] 安全審核報告已產出？
- [ ] 負面案例測試通過率 100%？
- [ ] 命中高風險字眼時，狀態正確切換為 PENDING 審核？
