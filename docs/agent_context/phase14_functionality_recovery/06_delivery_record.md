```
主流程
  │
  ▼
config/search_config.json
  │ provider: "gemini_fewshot"  ← 改這個切換策略
  ▼
ConfigDrivenSearchTool
  │
  ▼
SearchToolFactory
  │
  ▼
返回 SearchResult
```

### 使用方式
```python
# 主流程（不需改動）
from src.services.config_driven_search import search
result = search("澳霸有限公司")  # 自動根據配置執行

# 切換策略
# 只需修改 config/search_config.json
{ "search": { "provider": "tavily" } }  // 快速
{ "search": { "provider": "gemini_fewshot" } }  // 完整
```
