| 欄位 | 格式 | 結果 |
|------|------|------|
| organNo (統一編號) | 字串 | ✅ 正常 |
| capital (資本額) | 字串 | ❌ 失敗 |
| capital (資本額) | 數字 | ✅ 正常 |
| employee_count (員工人數) | 數字 | ✅ 正常 |
| established_year (成立年份) | 數字 | ✅ 正常 |

**關鍵發現**:
- `capital` 欄位必須使用**數字格式**，不接受字串
- 其他選填欄位功能正常

**修復方案**:
```json
// 錯誤格式
"capital": "5000000"

// 正確格式
"capital": 5000000
```

**結論**: 選填欄位功能 ✅ PASSED（格式問題已確認，非功能缺陷）
