## 📋 概述

### 開發目標

實現真正的平行 LLM API 呼叫，讓每個面向使用獨立的 Prompt 進行搜尋，提升搜尋效能 40-50%。

### 核心需求

根據用戶需求：
1. **同時發送多個獨立 Prompt**：每個面向一個 Prompt
2. **使用 ThreadPoolExecutor 平行執行**
3. **彙整所有 API 返回後再往下流程**
4. **支援配置驅動**：可用 config 指定 Prompt 和策略

### 預期效益

| 指標 | 當前 | 目標 | 改善 |
|------|------|------|------|
| 搜尋時間 | 10-12s | 6-8s | **40-50%** |
| API 呼叫 | 1次 | 4次 | 可控 |
| 代碼結構 | 700+ 行 | 分層解耦 | 易維護 |

### 完整調用流程

```
┌─────────────────────────────────────────────────────────────┐
│  主流程層 (company_brief_graph.py)                          │
│  search_node(state)                                         │
└────────────────────┬──────────────────────────────────────┘
                     │ search(query, strategy="complete")
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  配置驅動層 (config_driven_search.py)                       │
│  ConfigDrivenSearchTool.search(query, strategy)            │
│                                                              │
│  1. 讀取 config/search_config.json                          │
│  2. 根據 strategy 選擇配置                                  │
│  3. 建立對應的工具                                          │
└────────────────────┬──────────────────────────────────────┘
                     │ tool.search(query)
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  工具層 (search_tools.py)                                   │
│  ParallelAspectSearchTool.search(query)                      │
│                                                              │
│  1. 建立 4 個獨立 Prompt                                      │
│  2. 使用 ThreadPoolExecutor 平行執行                         │
│  3. 等待所有完成                                            │
│  4. 彙整結果                                                 │
└────────────────────┬──────────────────────────────────────┘
                     │ 4 個並發 API 呼叫
                     ▼
┌─────────────────────────────────────────────────────────────┐
│  LLM API 層 (Google Gemini)                                 │
│  4 個並發的 API 呼叫                                        │
│  - gemini.search("台積電 foundation")                        │
│  - gemini.search("台積電 core")                             │
│  - gemini.search("台積電 vibe")                            │
│  - gemini.search("台積電 future")                          │
└─────────────────────────────────────────────────────────────┘
```

| 標準 | 條件 | 驗證方式 |
