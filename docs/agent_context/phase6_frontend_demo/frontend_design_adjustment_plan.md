## 3. 差異分析

### 3.1 佈局順序差異

```
現有設計（桌面版）:     參考設計:
┌─────────┬─────────┐   ┌─────────────────────┐
│  表單   │  結果   │   │       結果         │
│ (左側)  │ (右側)  │   │      (上方)        │
└─────────┴─────────┘   ├─────────────────────┤
                       │       表單           │
                       │      (下方)          │
                       └─────────────────────┘
```

**差異說明**：
- 現有：左右兩欄，表單在左，結果在右
- 參考：上下堆疊，結果在上，表單在下

### 3.2 行動版順序差異

```
現有設計（手機版）:     參考設計:
┌─────────────┐        ┌─────────────┐
│    結果     │        │    結果     │
│  (先顯示)   │        │  (先顯示)   │
├─────────────┤        ├─────────────┤
│    表單     │        │    表單     │
│  (後顯示)   │        │  (後顯示)   │
└─────────────┘        └─────────────┘
```

**說明**：行動版順序相同，都是結果優先，但參考設計沒有明顯的搜尋 Bar，而是改為表單內的輸入區域。

### 3.3 視覺風格差異

| 項目 | 現有設計 | 參考設計 |
|------|----------|----------|
| 標題文字 | 較長，含英文 | 較短，中文為主 |
| 間距 | 使用 `py-6`, `gap-6` | 更緊湊 |
| 搜尋 Bar | 無 | 有（頂部） |
| 區塊分隔 | 明確的卡片邊界 | 較不明顯 |
| 背景 | 灰色背景 (`bg-gray-50`) | 可能較白 |

### 3.4 功能性差異

| 功能 | 現有 | 參考（推測） |
|------|------|--------------|
| 搜尋功能 | 無 | 可能有快速搜尋 |
| 結果預覽 | 完整顯示 | 可能較精簡 |
| 表單驗證 | 有 | 可能簡化 |

## 6. 技術細節

### 6.1 App.vue 修改重點

```vue
<!-- 現有結構 -->
<div class="grid grid-cols-1 lg:grid-cols-2 gap-6">
  <div class="order-2 lg:order-1">
    <BriefForm :loading="loading" @submit="handleSubmit" />
  </div>
  <div class="order-1 lg:order-2">
    <ResultPanel :result="result" :loading="loading" />
  </div>
</div>

<!-- 調整後結構（垂直堆疊） -->
<div class="flex flex-col gap-4">
  <!-- 結果區塊在上方 -->
  <div>
    <ResultPanel :result="result" :loading="loading" />
  </div>
  <!-- 表單區塊在下方 -->
  <div>
    <BriefForm :loading="loading" @submit="handleSubmit" />
  </div>
</div>
```

### 6.2 標題調整

```vue
<!-- 現有 -->
<h1 class="text-2xl font-bold text-gray-900">
  Organ Brief Optimization
</h1>
<span class="text-sm text-gray-500">公司簡介生成/優化工具</span>

<!-- 調整後 -->
<h1 class="text-xl font-bold text-gray-900">
  1111 人力銀行 - 公司簡介生成工具
</h1>
```

### 6.3 間距調整

| 元件 | 現有值 | 建議值 |
|------|--------|--------|
| Header py-4 | py-4 | py-3 |
| Main py-6 | py-6 | py-4 |
| 區塊 gap-6 | gap-6 | gap-4 |
| Card padding | p-6 | p-4 |
